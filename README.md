# **AlrtAgent — Autonomous SOC Alert Triage Agent**

AlrtAgent is an AI-native security agent that receives SIEM alerts, enriches IOCs via AbuseIPDB and VirusTotal, correlates multi-event attack sequences, and generates analyst-grade investigation reports autonomously.

## How It Works

1. Alert ingested via FastAPI webhook or simulator.
2. Agent classifies alert type and extracts key fields.
3. Source IP enriched via AbuseIPDB and VirusTotal APIs.
4. Log context searched for correlated events.
5. Attack patterns identified across the event sequence.
6. MITRE ATT&CK techniques retrieved via RAG.
7. Full investigation report generated by LLM.
